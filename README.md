# AnswerAI - White labeled version of OpenWebUI

This branch contains a white-labeled version of OpenWebUI, rebranded as AnswerAI. The following changes have been made:

1. Rebranding from "Open WebUI" to "AnswerAI" throughout the codebase
2. Removing the call functionality
3. Removing chat controls functionality from the UI
4. Modifying the general settings to rename "WebUI setting" to "Settings"
5. Removing the language selection from the general settings
6. Making the About section blank
7. Removing version text from the message input bar
8. Updating URLs from "openwebui.com" to "answerai.in"

The changes ensure that the application remains fully functional while meeting the branding requirements of AnswerAI.
